## Description

Cheyni Backend TypeScript repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run dev

# production mode
$ npm run start
```

Don't forget to get the ENV credentials on ENV.EXAMPLE
